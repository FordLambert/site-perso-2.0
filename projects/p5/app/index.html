Random quotes Javascript generator

This is a little project that I made during my OpenClassroom course :
It was my first Javascript project.

The goal was to create a little generator of random quotes in pure Javascript.
It take three (random) parts (beginning, middle and end) and make a sentence with it.

After a first procedural version, my mentor and I decided that it will be a nice training to do it object oriented instead, as a first step into this way of coding with little code for easier comprehension.
User interface was not a requirement (console logs was enough) but I like my presentation to look nice (obviously not a graphist work but you get the idea) so I made this little Bootstrap page.
